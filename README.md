# APIcasso Brush


## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'apicasso_brush'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install apicasso_brush
```
